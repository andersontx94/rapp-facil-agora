# RAPP Fácil Agora

Projeto web criado para ajudar no preenchimento do RAPP (IBAMA), deixando o processo mais simples e organizado para quem precisa lidar com esse relatório no dia a dia.

A ideia surgiu da dificuldade comum que empresas e profissionais têm com formulários extensos e pouco intuitivos, o que acaba gerando erros e retrabalho.

## Sobre o projeto
Este projeto começou com a criação da interface usando a plataforma Lovable.  
Após isso, o código foi exportado e passou a ser mantido manualmente, com ajustes, melhorias e organização diretamente no GitHub.

O deploy foi feito utilizando a Vercel, com ambiente em produção e domínio configurado.

## O que foi feito
- Criação da interface inicial
- Exportação do projeto para código
- Ajustes e manutenção diretamente no GitHub
- Organização da estrutura do projeto
- Versionamento do código
- Deploy em produção com Vercel
